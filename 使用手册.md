# Science Input Helper 中文使用手册

## 1. 插件定位

Science Input Helper 是一个 Obsidian 理科笔记输入辅助插件，适合中文高中到本科低年级课程笔记。

它主要补充 Latex Suite：

- Latex Suite：负责完整数学 LaTeX 快捷输入、矩阵、复杂环境和 snippets。
- Science Input Helper：负责化学式、单位、常见理科公式模板和中文符号快速输入。

本插件不接入 AI，也不提供完整 snippets 系统。

## 2. 30 秒快速上手

1. 在 Markdown 笔记中按 `Ctrl + Shift + Q` 打开快速输入框。
2. 输入 `化 H2SO4`，按 `Enter` 插入行内公式。
3. 输入 `单位 9.8m/s2`，按 `Enter` 插入单位。
4. 输入 `物 动能`，查看“插入前预览”后按 `Enter`。
5. 输入 `可逆`，插入 `\rightleftharpoons`。
6. 输入 `help` 查看前缀帮助。
7. 按 `Esc` 退出。

## 3. 快速输入模式

快速输入框会显示“插入前预览”。如果有候选结果，可以用键盘选择：

- `Enter`：插入默认格式。多数理科表达式会插入为行内公式 `$...$`
- `Ctrl + Enter`：插入块公式
- 注意：`raw`、Unicode 希腊字母和罗马数字会作为普通文本插入，不自动包裹 `$...$`
- `Tab`：应用第一候选或当前候选
- `ArrowUp / ArrowDown`：切换候选
- `Esc`：退出

输入框可以移动：按住 `Science Input` 标题栏拖动即可。拖动后的位置会被记住。设置页里可以重置位置。

## 4. 快速输入前缀

中文前缀：

```text
化 H2SO4              化学式
单位 9.8m/s2          单位
物 动能               物理模板
化学 pH               化学模板
生化 ATP水解          生物化学模板
原 E=mc^2             原样 LaTeX
希 阿尔法             希腊字母
罗马 12               罗马数字
```

英文前缀：

```text
c H2SO4
u 9.8m/s2
p 动能
ch pH
b ATP水解
raw E=mc^2
g alpha
r 12
roman 12
```

输入 `help`、`帮助` 或 `?` 会显示前缀帮助。

## 5. 化学式转换

推荐在快速输入模式中使用 `化` 或 `c` 前缀，例如 `化 H2SO4`。无前缀输入时，插件会尝试自动识别，但复杂内容建议使用前缀，以减少和生物化学模板、普通 raw 输入之间的误判。

```text
H2O                -> \mathrm{H_2O}
Ca(OH)2           -> \mathrm{Ca(OH)_2}
NH4+              -> \mathrm{NH_4^{+}}
CH3COOH           -> \mathrm{CH_3COOH}
CH3COO-           -> \mathrm{CH_3COO^{-}}
SO4 2-            -> \mathrm{SO_4^{2-}}
CuSO4·5H2O        -> \mathrm{CuSO_4\cdot 5H_2O}
```

反应符号：

```text
->     -> \rightarrow
<->    -> \rightleftharpoons
down   -> \downarrow
up     -> \uparrow
heat   -> \xrightarrow{\Delta}
```

## 6. 反应条件箭头

Science Input Helper 支持在反应箭头或可逆箭头上方、下方添加条件、催化剂、温度等内容。

### 可逆箭头

```text
可逆 -> \rightleftharpoons
可逆[加热] -> \overset{\Delta}{\rightleftharpoons}
可逆[加热][浓硫酸] -> \overset{\Delta}{\underset{\mathrm{浓硫酸}}{\rightleftharpoons}}
可逆 上 加热 下 浓硫酸 -> \overset{\Delta}{\underset{\mathrm{浓硫酸}}{\rightleftharpoons}}
eq[heat][H2SO4] -> \overset{\Delta}{\underset{\mathrm{H_2SO_4}}{\rightleftharpoons}}
```

### 生成箭头

```text
生成[加热] -> \xrightarrow{\Delta}
生成[光照] -> \xrightarrow{h\nu}
生成[加热][催化剂] -> \xrightarrow[\mathrm{cat.}]{\Delta}
ra[heat][cat] -> \xrightarrow[\mathrm{cat.}]{\Delta}
```

### 常见条件

```text
加热 / heat -> \Delta
光照 / hv / hν -> h\nu
催化剂 / cat -> \mathrm{cat.}
浓硫酸 -> \mathrm{浓硫酸}
H2SO4 -> \mathrm{H_2SO_4}
MnO2 -> \mathrm{MnO_2}
170℃ -> 170\,^{\circ}\mathrm{C}
25°C -> 25\,^{\circ}\mathrm{C}
```

### 完整反应式

```text
反应 CH3COOH + C2H5OH 可逆[加热][浓硫酸] CH3COOC2H5 + H2O
```

输出：

```latex
\mathrm{CH_3COOH + C_2H_5OH} \overset{\Delta}{\underset{\mathrm{浓硫酸}}{\rightleftharpoons}} \mathrm{CH_3COOC_2H_5 + H_2O}
```

说明：
- 本功能只负责排版，不判断反应是否真实、条件是否正确或方程式是否配平。
- 完整反应式转换目前属于简单实验功能，复杂反应式建议转换后人工检查。

## 7. 单位格式化

```text
9.8m/s2             -> 9.8\,\mathrm{m/s^2}
1e-3 mol/L          -> 1\times10^{-3}\,\mathrm{mol/L}
37°C                -> 37\,^{\circ}\mathrm{C}
8.314J/(mol K)      -> 8.314\,\mathrm{J/(mol\,K)}
```

单位格式化只改变显示格式，不进行单位换算。例如不会把 `1000mg` 自动换算成 `1g`。

## 8. 公式模板和特殊符号

常用模板：

```text
物 动能       -> E_k=\frac{1}{2}mv^2
化学 pH       -> \mathrm{pH}=-\lg[H^+]
生化 ATP水解  -> \mathrm{ATP + H_2O \rightarrow ADP + P_i + H^+}
```

关系符号：

```text
远大于     -> \gg
远小于     -> \ll
约等于     -> \approx
成正比     -> \propto
不等于     -> \ne
大于等于   -> \ge
小于等于   -> \le
```

推理符号：

```text
推出       -> \Rightarrow
等价       -> \Leftrightarrow
因为       -> \because
所以       -> \therefore
```

化学/反应符号：

```text
可逆       -> \rightleftharpoons
沉淀/down  -> \downarrow
气体/up    -> \uparrow
加热/heat  -> \xrightarrow{\Delta}
```

## 9. 希腊字母输入

支持直接输入英文名或中文名：

| 输入 | LaTeX 输出 | Unicode 输出 |
| --- | --- | --- |
| `alpha` / `阿尔法` | `\alpha` | `α` |
| `g alpha` / `希 阿尔法` | `\alpha` | `α` |
| `delta` | `\delta` | `δ` |
| `Delta` / `大写delta` / `大写德尔塔` | `\Delta` | `Δ` |
| `omega` / `欧米伽` | `\omega` | `ω` |
| `varphi` / `变体phi` | `\varphi` | `φ` |

输入下面任意关键词会显示全部希腊字母候选：

```text
希腊字母
greek
g
```

单独输入 `g` 会显示全部希腊字母候选；输入 `g alpha`、`g delta` 等会搜索对应希腊字母。

候选列表会显示 Unicode 符号、英文名、中文名和即将插入的 LaTeX 或 Unicode 内容。

注意：大写希腊字母使用显式映射表，不会简单把首字母大写拼成 `\Alpha` 这类通常不适合 MathJax 的写法。

## 10. 罗马数字转换

罗马数字必须带前缀，不会裸数字自动转换：

```text
12       不会自动转换成 XII
r 12     -> XII
roman 12 -> XII
罗马 12  -> XII
```

罗马数字默认作为普通文本插入，不自动加 `$...$`。

罗马数字主要用于章节编号、题号、化合价标记等场景，因此默认不作为数学公式插入。

| 输入 | 输出 |
| --- | --- |
| `r 1` | `I` |
| `r 4` | `IV` |
| `r 9` | `IX` |
| `r 12` | `XII` |
| `r 49` | `XLIX` |
| `r 944` | `CMXLIV` |
| `r 2024` | `MMXXIV` |
| `r 3999` | `MMMCMXCIX` |

支持范围是 1 到 3999。`r 0`、`r -1` 会提示无效；`r 4000` 会提示超出范围。

## 11. 数学结构包裹

Science Input Helper 支持常见数学结构的快速包裹，适合把已经输入的长式子变成根号、分式、括号、绝对值等 LaTeX 结构。

### 根号

```text
根号 b^2-4ac -> \sqrt{b^2-4ac}
三次根号 x+1 -> \sqrt[3]{x+1}
sqrt3 x+1 -> \sqrt[3]{x+1}
n次根号 x+1 -> \sqrt[n]{x+1}
```

### 分式

```text
分式 a+b / c+d -> \frac{a+b}{c+d}
frac x^2+2x+1 / x-1 -> \frac{x^2+2x+1}{x-1}
数 分式 a+b / c+d -> \frac{a+b}{c+d}
m frac a+b / c+d -> \frac{a+b}{c+d}
```

也可以先选中表达式，再执行命令：

```text
Science Input Helper: Wrap selection as fraction numerator
```

### 括号和绝对值

```text
括号 x+1 -> \left(x+1\right)
方括号 x+1 -> \left[x+1\right]
大括号 x+1 -> \left\{x+1\right\}
绝对值 x-1 -> \left|x-1\right|
范数 v -> \left\|v\right\|
```

### 向量、上标、下标

```text
向量 v -> \vec{v}
线段 AB -> \overrightarrow{AB}
上标 n+1 -> ^{n+1}
下标 i=1 -> _{i=1}
```

说明：

- 数学结构转换只做格式包裹，不检查数学表达式是否正确。
- 复杂分式只按第一版简单规则处理，例如 `a/b/c -> \frac{a}{b/c}`，不做完整数学语法解析。
- 复杂数学输入仍推荐配合 Latex Suite 使用。

## 12. 设置页说明

- `Enable Greek letter helper`：启用或关闭希腊字母输入。
- `Greek output mode`：选择 LaTeX 或 Unicode 输出。LaTeX 模式输出 `\alpha`；Unicode 模式输出 `α`，并作为普通文本插入。
- 当 `Greek output mode` 设置为 Unicode 时，即使按 `Enter`，也不会自动包裹 `$...$`。
- `Include variant Greek letters`：是否包含 `\varepsilon`、`\varphi`、`\vartheta`、`\varrho`、`\varsigma`。
- `Show all Greek letters for "希腊字母" / "greek"`：是否在输入 `希腊字母`、`greek` 或 `g` 时显示全部希腊字母候选。

## 13. 与 Latex Suite 的推荐分工

| 任务 | 推荐工具 |
| --- | --- |
| 数学 snippets、矩阵、复杂环境 | Latex Suite |
| 化学式、单位、中文模板搜索、希腊字母、罗马数字 | Science Input Helper |
| 完整 LaTeX 语法补全 | Latex Suite |
| 复杂公式转换后校对 | 人工检查 + Latex Suite |

## 14. 插件限制

- 不检查科学内容正确性。
- 不自动配平化学方程式。
- 反应条件箭头只做排版，不判断条件是否真实或是否适用于该反应。
- 完整反应式转换目前是简单实验功能，不解析复杂有机结构式、机理箭头或多步反应。
- 不进行单位换算。
- 数学结构转换只做格式包裹，不判断数学含义。
- 分式转换第一版只处理简单的 `A / B` 结构，不解析复杂嵌套表达式。
- 不替代 Latex Suite。
- 不保证所有复杂化学式、复杂离子和有机结构式都能一次转换正确。
- 复杂公式转换后需要人工检查。
- 复杂数学公式仍建议使用 Latex Suite 或手动校对。
