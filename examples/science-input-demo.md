# Science Input Helper 示例笔记

## 乙酸乙酯的制备

反应式：

$$
\mathrm{CH_3COOH + C_2H_5OH \rightleftharpoons CH_3COOC_2H_5 + H_2O}
$$

带条件的可逆反应：

```text
反应 CH3COOH + C2H5OH 可逆[加热][浓硫酸] CH3COOC2H5 + H2O
```

$$
\mathrm{CH_3COOH + C_2H_5OH} \overset{\Delta}{\underset{\mathrm{浓硫酸}}{\rightleftharpoons}} \mathrm{CH_3COOC_2H_5 + H_2O}
$$

产率计算：

$$
\eta = \frac{m_{\text{实际}}}{m_{\text{理论}}} \times 100\%
$$

数学结构快速输入：

```text
分式 m实际 / m理论 -> \frac{m实际}{m理论}
上标 n+1           -> ^{n+1}
括号 x+1           -> \left(x+1\right)
```

常用单位：

$1.0\,\mathrm{mol/L}$

$1.0\,\mathrm{g/mL}$

## 离子反应

沉淀反应：

$$
\mathrm{Ca^{2+} + CO_3^{2-} \rightarrow CaCO_3 \downarrow}
$$

常见离子：

```text
化 NH4+       -> \mathrm{NH_4^{+}}
化 OH-        -> \mathrm{OH^{-}}
化 NO3-       -> \mathrm{NO_3^{-}}
化 SO4 2-     -> \mathrm{SO_4^{2-}}
化 CH3COO-    -> \mathrm{CH_3COO^{-}}
```

## 普通物理：波动

波速：

$$
v=\lambda f
$$

根号结构：

```text
根号 b^2-4ac -> \sqrt{b^2-4ac}
三次根号 x+1 -> \sqrt[3]{x+1}
```

声强级：

$$
L=10\lg\frac{I}{I_0}
$$

常用符号：

```text
希 lambda -> \lambda
约等于    -> \approx
成正比    -> \propto
远大于    -> \gg
```
